# Playwright + TypeScript

Proyecto de automatización de pruebas utilizando Playwright y TypeScript.

## Instalación

Clonar el repositorio e instalar dependencias:

```bash
npm install
```

## Instalación de navegadores

Instalar los navegadores necesarios para Playwright:

```bash
npx playwright install
```

## Ejecutar tests - headed

```bash
npx playwright test --headed
```

## Ejecutar tests - headless

```bash
npx playwright test
```

## Ver reporte

```bash
 npx playwright show-report
```
